# Pre-Entrega 1 - Primeros pasos con JavaScript

Proyecto realizado para la primera pre-entrega del curso JavaScript Flex.

## Descripción

Este proyecto consiste en un script interactivo básico que solicita datos al usuario mediante `prompt()`, procesa la información con variables, operadores aritméticos y conversión de tipos, y muestra el resultado mediante `alert()` y `console.log()`.

La práctica permite aplicar los primeros conceptos de JavaScript: declaración de variables con `const` y `let`, tipos de datos, conversión de valores ingresados por el usuario, operaciones matemáticas simples y comunicación de resultados.

## Funcionalidades

- Solicita el nombre del usuario.
- Solicita el año de nacimiento.
- Solicita el tipo de proyecto web a cotizar.
- Solicita la cantidad de páginas del sitio.
- Convierte datos ingresados por `prompt()` a tipo número.
- Calcula la edad aproximada del usuario.
- Calcula un presupuesto estimado.
- Muestra el resultado final mediante `alert()`.
- Imprime los datos procesados en consola con `console.log()`.

## Tecnologías utilizadas

- HTML5
- JavaScript

## Conceptos aplicados

- Variables modernas con `const` y `let`.
- Uso de `prompt()` para capturar datos.
- Uso de `Number()` para conversión de tipos.
- Operadores aritméticos.
- Concatenación de strings.
- Uso de `alert()` para mostrar resultados al usuario.
- Uso de `console.log()` para mostrar información en consola.
- Vinculación de un archivo JavaScript externo con `defer`.
